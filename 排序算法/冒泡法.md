## 冒泡法

### 稳定性：稳定排序算法

### 平均时间复杂度：O(N2)

---
```java
public void bubbleSort(int[] arr){
        int length=arr.length;
        for(int i=0;i<length-1;i++){
            //每一轮循环都把最后一个数排序好(将最大的数放置数组最后面)
            for(int j=0;j<length-i-1;j++){
                if(arr[j-1]>arr[j]){
                    //交换操作
                    int temp=arr[j];
                    arr[j]=arr[j-1];
                    arr[j-1]=temp;
                }
            }
        }
    }
```

### 优化方案：当遍历不发生交换时，证明排序已经结束，此时不再需要交换

```java
public void bubbleSort(int[] arr){
        int length=arr.length;
        for(int i=0;i<length-1;i++){
             boolean flag=false;
            //每一轮循环都把最后一个数排序好(将最大的数放置数组最后面)
            for(int j=0;j<length-i-1;j++){
                if(arr[j-1]>arr[j]){
                    //交换操作
                    int temp=arr[j];
                    arr[j]=arr[j-1];
                    arr[j-1]=temp;
                    flag=true;
                }
                if(flag==false) break;
            }
        }
    }
```
